# Dashboard de Atletismo — Comitê Paralímpico Brasileiro

Painel de análise de desempenho de atletas desenvolvido em Power BI,
com base em dados fornecidos pelo Comitê Paralímpico Brasileiro.

![image_alt](https://github.com/JulliaCarneiro/dashboard-atletismo-cpb/blob/main/repositorio-projeto/images/visao_geral.png?raw=true)

## Painéis

| Painel | Descrição |
|---|---|
| Visão Geral dos Atletas | Resumo geral com perfil e métricas dos atletas |
| Ranking - Corrida | Classificação por posição e tempo de cada atleta |
| Ranking - Salto em Distância | Classificação por posição e distância de cada atleta |
| Ranking - Arremesso de Peso | Classificação por posição e distância de cada atleta |
> **Dica:** Para utilizar o filtro, use **CTRL + clique** para abrir e fechar nos ícones.

## Arquivos

| Arquivo | Descrição |
|---|---|
| `dashboard/` | Arquivo .pbix do Power BI |
| `base_de_dados/` | Base de dados original em Excel |
| `docs/dicionario_de_dados.md` | Descrição das colunas e métricas |

## Como abrir

1. Instale o [Power BI Desktop](https://powerbi.microsoft.com/)
2. Abra o arquivo dentro da pasta `dashboard/`
3. Se necessário, atualize o caminho da fonte de dados em
**Transformar Dados > Configurações da Fonte**

## Fonte de Dados

Base fornecida pelo **Comitê Paralímpico Brasileiro** contendo
dados de desempenho dos atletas nas modalidades de corrida,
salto em distância e arremesso de peso.
